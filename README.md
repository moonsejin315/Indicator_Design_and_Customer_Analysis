- **개인 학습**: 수업 내용을 바탕으로 추가 학습한 개념, 데이터 분석 실습, 인사이트 공유
- **지속적인 개선**: 수업 종료 후에도 실무 혹은 포트폴리오 관점에서의 분석 추가 예정
